Instalación y Ejecución
Requisitos previos
Tener instalado Visual Studio Code con soporte para C++.
Instalar la biblioteca raylib.
Descarga y preparación
Descomprimir los archivos correspondientes a raylib y al proyecto Tower Defense en ubicaciones accesibles dentro del sistema.
Ejecución del proyecto
Acceder al directorio del proyecto Tower Defense.
Ejecutar el archivo TowerDefense.exe.
Advertencia de seguridad en Windows
En caso de que el sistema muestre una advertencia indicando que la aplicación no es segura:
Seleccionar la opción "Más información".
Posteriormente, hacer clic en "Ejecutar de todos modos" para continuar con la ejecución.
