## Instalación y Ejecución

### 1. Requisitos previos
- Tener instalado Visual Studio Code con soporte para C++.
- Instalar la biblioteca raylib.

### 2. Descarga y preparación
- Descomprimir los archivos de raylib y del proyecto Tower Defense en el sistema.

### 3. Ejecución del proyecto
- Acceder a la carpeta del proyecto Tower Defense.
- Ejecutar el archivo `TowerDefense.exe`.

### 4. Advertencia de seguridad en Windows
Si aparece una advertencia indicando que la aplicación no es segura:
1. Seleccionar la opción **"Más información"**.
